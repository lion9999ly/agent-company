## 关键设计约束总结

### 户外骑行核心约束

| 约束 | 阈值 | 原因 |
|------|------|------|
| **白天可读亮度** | ≥1000 nits到眼 | 正午阳光10000lux环境对比度需求 |
| **夜间不刺眼** | 可调亮度100-300nits | 夜间道路照明不足 |
| **眼盒稳定性** | ≥8×6mm有效 | 头盔佩戴姿态变化大 |
| **续航** | ≥8小时 | 长途骑行需求 |
| **重量** | HUD模组≤80g | 整盔≤1.65kg |
| **透过率** | ≥70% | 安全视野需求 |
| **抗振动** | ISO 16750-3标准 | 摩托车振动环境 |

### 方案推荐

| V1量产方案 | OLED + FreeForm | 原因 |
|-----------|-----------------|------|
| 亮度 | 500-800nits够用但偏弱 | 白天勉强可读 |
| 眼盒 | 10×8mm较宽松 | 头盔姿态容忍度高 |
| 成本 | $80-180 | 商业可行 |
| 成熟度 | 高 | 供应链稳定 |
| 建议 | 单目、单色/有限彩色、信息密度低 | 先验证市场 |

| V2预研方案 | MicroLED + 树脂波导 | 原因 |
|-----------|---------------------|------|
| 亮度 | 800-2000nits | 白天清晰可读 |
| 眼盒 | 8×6mm偏小 | 需验证头盔适配 |
| 成本 | $180-450 | 高端定价支撑 |
| 成熟度 | 中 | 需验证量产一致性 |
| 建议 | 单色绿光先行、全彩待彩虹效应解决 | 技术储备 |
