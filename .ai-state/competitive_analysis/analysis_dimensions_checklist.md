## 使用方法

### 分析前检查

```
□ 已加载本维度清单
□ 已确认产品名称（必须先验证官方信息）
□ 已准备数据采集工具
```

### 分析中要求

```
□ 每个维度必须填写或标注"待采集"
□ 每个数据点必须标注置信度（high/medium/low）
□ 缺失数据必须标注原因
```

### 分析后验证

```
□ 30个维度是否全部覆盖？
□ 数据来源是否标注？
□ 置信度是否标注？
□ 待采集项是否列出下一步行动？
```

---

## 数据置信度定义

| 等级 | 来源类型 | 示例 |
|------|----------|------|
| **high** | 官方数据 | 官网规格、产品包装、官方声明 |
| **medium** | 权威报道 | 科技媒体评测、行业报道 |
| **low** | 估计/推断 | 基于同类产品推断 |
| **待采集** | 未获取 | 需要进一步采集的数据 |
