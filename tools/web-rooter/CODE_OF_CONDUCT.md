# Code of Conduct

## Our Commitment

We aim to keep this project open, respectful, and practical for all contributors.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical facts and reproducible evidence.
- Assume good intent and discuss disagreements professionally.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Discrimination of any kind.
- Publishing private/sensitive information without consent.

## Enforcement

Maintainers may edit, remove, or reject contributions that violate this policy.
