# HUD Demo 生成规范 v2

## 物理布局模型（不可违反）

面镜两侧各有一条**竖向显示带**，中间完全透明（路面视野）。

```
┌─────────────────────────────────────────────────┐
│ ┌──────┐                           ┌──────┐     │
│ │  LT  │                           │  RT  │     │
│ │      │                           │      │     │
│ ├──────┤                           ├──────┤     │
│ │ 导航 │     中央透明区域          │      │     │
│ │ 地图 │     （路面可见）          │      │     │
│ ├──────┤                           ├──────┤     │
│ │  LB  │                           │  RB  │     │
│ │      │                           │      │     │
│ └──────┘                           └──────┘     │
│          ←── 辐辏安全间距 ──→                    │
└─────────────────────────────────────────────────┘
```

### 布局硬约束

1. 信息只能出现在左右两条竖向显示带内（四角 LT/RT/LB/RB + 左侧中段导航区）
2. 中央区域必须完全透明——不得放置任何 HUD 信息
3. 中央间距满足辐辏约束：左眼显示内容右眼不可见（默认间距 ≥ 人因自然约束，做成可调参数）
4. 显示带位于面镜高度中偏上位置
5. 显示区域的大小、位置必须提供可调控件（sandbox 面板中）

### 路面背景

- 三种模式均为半透明叠加（包括 FreeForm）
- 默认显示深色路面背景，用户可导入自定义图片替换
- 透明度用户可调（模拟光致变色面罩效果）
- 图片为静态

## 三种光学模式

### 共同点
- 布局结构相同（两条竖向显示带 + 中央透明）
- 差异体现在 FOV 大小、亮度、色彩能力

### 模式 A：OLED + FreeForm 棱镜
- FOV: 38°（可调参数）
- 色彩: 全彩 RGB
- 亮度: 高（到眼 ~10,000 nits 单绿 / ~1,500 nits 全彩）
- 对比度: 50,000:1
- 信息编码: 颜色 + 亮度 + 大小 + 位置 + 闪烁
- 背景: 半透明叠加路面（反射率 ~50%，可调）

### 模式 B：全彩光波导
- FOV: 28-35°（可调参数）
- 色彩: 全彩 RGB
- 亮度: 低（到眼 <1,000 nits）
- 透过率: 70-85%（背景更清晰）
- 信息编码: 同模式 A
- 背景: 半透明叠加路面（透过率更高，可调）

### 模式 C：单绿光波导
- FOV: 28-35°（可调参数）
- 色彩: **唯一色相 #00FF00（520nm 绿光）**
- 亮度: 中（到眼 ~1,500 nits）
- **视觉编码严格规则（不可违反）：**
  - 不允许任何其他颜色，包括深浅绿变体
  - 信息区分只能使用四种手段：亮度百分比、字号大小、位置、闪烁频率
  - 亮度分级为可调参数，默认值：
    - 紧急预警: 100% + 闪烁
    - 主要信息（速度/导航）: 80%
    - 次要信息: 50%
    - 背景/边框: 20%
- 切换到单绿模式时，所有颜色编码必须自动转换为亮度编码，用户必须能看到明显差异

## 信息区域分配

| Zone | 位置 | 主要内容 | 次要内容 |
|------|------|---------|---------|
| LT | 左上 | 速度（km/h）+ 速度等级色 | 巡航/警告状态标签 |
| RT | 右上 | 交通标志（持续显示到下一个）| 状态信息（电量/信号/温度）|
| LB | 左下 | ADAS 预警方向 + LDW 虚线 | 来电接听/拒绝 |
| RB | 右下 | ADAS 预警详情 | Mesh 组队信息 |
| 左中段 | 左侧竖条中间 | 导航地图（可调显示级别）| — |

## ADAS 预警规范

### 预警类型（全部体现）
- FCW（前碰撞预警）
- BSD（盲区检测）
- Dooring（开门杀）
- 行人检测
- LDW（车道偏离）
- 交通标志识别

### 每个预警必须显示
- 威胁类型名称
- 方向（前/左/右）
- 距离（米）
- 关闭速度（km/h）
- TTC（Time To Collision，秒，倒计时显示）
- 威胁等级：提示（1Hz 闪烁）/ 警告（2Hz）/ 紧急（4Hz）

### 方向映射
- 前方威胁 → LT + RT zone 同时闪烁
- 左后方来车（BSD）→ 左侧竖条**外边缘**闪烁
- 右侧威胁（Dooring）→ 右侧竖条**外边缘**闪烁
- LDW 左侧压线 → 左侧竖条**内边缘**虚线闪烁
- LDW 右侧压线 → 右侧竖条**内边缘**虚线闪烁

### 多重预警并发
- TTC 最短的威胁最大化显示（占主要 zone 面积）
- 其他威胁弱化显示（缩小，降低亮度）
- 不得隐藏任何已触发的预警

### 交通标志识别
- RT zone 持续显示识别到的标志（如限速 60）
- 直到识别到下一个标志才替换
- 无标志时显示系统状态（电量/信号）

## 状态机

### 7 种模式
cruise / nav / call / music / mesh / warn / dvr

### 优先级（高→低）
warn > call > nav > mesh > music > dvr > cruise

### 优先级抢占行为
- 高优先级触发时，低优先级信息**缩小到角落**，不消失
- 预警结束后**自动恢复**之前的状态，无需用户操作

### 速度分级
| 等级 | 速度范围 | 显示策略 |
|------|---------|---------|
| S0 | 0-30 km/h | 全部信息显示 |
| S1 | 31-60 km/h | 全部信息显示 |
| S2 | 61-100 km/h | 导航降级为简易地图，隐藏音乐/组队详情 |
| S3 | >100 km/h | 只保留速度 + ADAS + 简化导航箭头 |

## 导航显示

### 三级显示
1. **箭头模式**：转向箭头 + 距离
2. **简易地图**：路口简图 + 转向指示
3. **全信息地图**：完整路线地图

### 切换规则
- S2/S3 速度时自动降级为箭头或简易地图
- 用户可手动切换（sandbox 按钮）
- 导航地图显示在左侧竖条中段
- 显示级别做成可调参数

## 通信与媒体

### 来电
- 显示来电人名称 + 接听/拒绝选项
- 优先级仅低于 ADAS 预警

### Mesh 组队
- 结合地图显示队友相对位置（前方/后方 + 距离）
- 显示在线人数

### 音乐
- 曲名 + 艺术家
- S2 以上自动隐藏

## 录像（DVR）

- 开机后显示 REC 指示灯
- 语音标记"精彩瞬间"的模拟（字幕显示 "Moment saved!"）
- 开机时有视野位置确认流程（预留接口，具体交互待定）

## 语音交互

- 语音内容在**非可视区域**（画布底部或顶部）以字幕形式呈现
- 字幕内容包括：用户语音 → AI 识别 → 执行结果
- 每个剧本中必须包含至少 2 次语音交互演示

## 剧本（4 个）

### 1. commute（通勤，50s）
日常通勤场景。导航 → 来电 → 到达。包含语音导航切换。

### 2. emergency（紧急，45s）
密集 ADAS 预警场景。FCW → BSD → Dooring → 行人检测 → LDW。展示多重预警并发和 TTC 倒计时。

### 3. group（组队，55s）
摩友组队骑行。Mesh 组队 → DVR 录像 → 语音标记精彩瞬间 → 导航。

### 4. touring（摩旅，60s）**新增**
长距离巡航场景。风景巡航 → DVR 生活记录 → 语音标记 → 偶尔导航 → 交通标志识别 → 低速停靠拍照。展示"骑行座舱"的沉浸体验。

## Sandbox 控件要求

必须提供以下可调参数（sandbox 面板）：
- 光学模式切换（F1/F2/F3）
- FOV 大小滑块
- 显示区域位置/大小调节
- 中央间距调节
- 背景透明度调节
- 背景图片上传/替换
- 亮度分级调节（单绿模式下各级百分比）
- 导航显示级别切换
- 速度设置
- 各种 ADAS 预警触发按钮（含方向和距离参数）
- 剧本播放控制

## 技术约束

- 单文件 HTML（含 CSS + JS）
- DOM ID 命名：zone-lt, zone-rt, zone-lb, zone-rb, bottom-bar, sandbox, timeline, boot-overlay, optics-indicator, hud-viewport-left, hud-viewport-right, speed-value
- CSS 变量使用 clamp() 响应式
- 所有可调参数存储在全局配置对象中，sandbox 修改后实时生效
- window 上暴露：setMode, getMode, setSpeed, setOptics, emitWarning, emitEvent, playScenario

## 验收标准（不可省略）

1. 三种光学模式切换后，同一个 ADAS 预警的编码方式必须可见差异（全彩用颜色，单绿用亮度）
2. 单绿模式下不得出现 #00FF00 以外的任何色相
3. 中央区域必须完全透明，可见路面背景
4. ADAS 预警必须同时显示距离 + 关闭速度 + TTC
5. 多重预警并发时 TTC 最短的最大化，其他弱化但可见
6. BSD 左后方闪烁出现在左侧竖条外边缘，不是内边缘
7. LDW 虚线闪烁出现在对应侧竖条内边缘，不是外边缘
8. 优先级抢占后被抢占内容缩小但不消失
9. 预警结束后自动恢复之前状态
10. touring 剧本包含语音标记精彩瞬间
11. 语音交互以字幕形式出现在非可视区域
12. 路面背景透明度可调
13. 交通标志识别持续显示在 RT zone 直到下一个标志
14. 显示区域大小和位置可通过 sandbox 调节
